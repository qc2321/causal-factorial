Replication archive: Causal Inference in Conjoint Analysis:
Understanding Multidimensional Choices via Stated Preference Experiments
Jens Hainmueller, Daniel Hopkins, Teppei Yamamoto

Files:
immigrant.dta (data for immigration conjoint)
candidate.dta (data for candidate conjoint)
replication_codePA.do (Stata code file to generate estimates)
replication_codePA_plots.r (R code file to plot estimates)

Instructions:
1. Copy all files into the same directory
2. Run replication_codePA.do in Stata to generate the estimates
3. Run replication_codePA_plots.r in R to generate the plots as PDFs 

Note that the Stata code exports the estimates as txt files and the R code loads these for the plots.
